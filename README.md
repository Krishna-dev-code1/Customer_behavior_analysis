#  Customer Behavior Analysis

An end-to-end data analytics project analyzing retail customer purchasing behavior using Python, PostgreSQL, and Power BI.
The goal of this project is to extract business insights from customer shopping data and visualize trends that can help improve marketing strategies and customer engagement.


## Tools & Technologies
	•	Python – Data cleaning and exploratory data analysis
	•	Pandas / NumPy – Data manipulation and preprocessing
	•	PostgreSQL – SQL queries for business analysis
	•	Power BI – Interactive dashboard and visualizations
	•	Jupyter Notebook – Data analysis environment


## Project Workflow
	1.	Data Cleaning & Preparation
	•	Loaded dataset using Pandas
	•	Handled missing values and data transformations
	2.	Exploratory Data Analysis
	•	Analyzed customer demographics and purchase patterns
	•	Investigated relationships between discounts, subscriptions, and spending
	3.	SQL Business Analysis
	•	Revenue comparison by gender and age group
	•	Customer segmentation (New, Returning, Loyal)
	•	Top purchased products by category
	•	Impact of discounts and subscription behavior
	4.	Data Visualization
	•	Built an interactive Power BI dashboard to visualize key insights and trends.


## Key Business Questions
	•	What is the total revenue generated by male vs female customers?
	•	Which products have the highest average review ratings?
	•	Do subscribed customers spend more than non-subscribers?
	•	Which products receive the highest percentage of discounted purchases?
	•	What is the revenue contribution of each age group?
